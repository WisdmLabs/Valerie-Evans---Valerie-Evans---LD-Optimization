Main queue manager class for handling LearnDash WooCommerce enrollments.

Manages the processing of LearnDash course enrollments when purchased through WooCommerce. This class addresses performance issues that occur when processing large numbers of course enrollments simultaneously by implementing controlled queue processing and asynchronous enrollment handling.

Features:

  • Configurable limits for simultaneous course/group enrollments
  • Asynchronous processing of large orders via REST API
  • User notifications for large orders with potential delays
  • Administrative settings for queue and product processing limits
  • Support for both regular orders and subscription-based enrollments
